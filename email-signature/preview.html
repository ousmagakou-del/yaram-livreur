✅ Animation embarquée en base64 — le logo YARAM officiel pulse en boucle (12 frames, loop infini, 22 Ko). Aucune dépendance externe : le GIF voyage dans le mail lui-même.

Pour l'installer dans Apple Mail :
1. Ouvre ce fichier dans Safari
2. Sélectionne uniquement la signature (du logo Y jusqu'à 🌿)
3. Cmd + C
4. Mail → Préférences → Signatures → Nouvelle → Cmd + V
5. Décoche "Toujours utiliser ma police par défaut"

Gmail : même méthode, depuis ⚙️ Paramètres → Signature → coller.

Encoding fixé — les emojis et accents (é, è, à) s'affichent correctement grâce au <meta charset="UTF-8"> ajouté dans yaram-signature.html.
